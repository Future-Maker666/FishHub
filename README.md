# FishHub（持续更新ing）

有点简单，但很有意思。
这是演示站点 [摸鱼办](https://fish.tngeek.com).

> 服务端请移步：https://github.com/lewkamtao/FishHub_Server

## 起步

安装依赖

```bash
yarn install
```

## 运行

运行在本地

```bash
yarn dev
```

你需要运行下面命令进行打包编译

```bash
yarn build
```

Checkout the [deployment documentation](https://v3.nuxtjs.org/docs/deployment).
